# 웹사이트 모음 – 개발 유틸리티

이 문서는 **개발할 때 쓰기 편한 온라인 유틸리티 사이트**를 정리합니다.

> 포함 기준
>
> - 개발 환경 설정, 포맷 변환, 인코딩/디코딩, 테스트 등에 도움 되는 도구
> - 설치 없이 웹에서 바로 사용할 수 있는 서비스

---

## 🔧 리스트

- [Env Converter](https://envconverter.matiboux.com/)

  - **설명**: `.env` 파일과 다른 포맷(JSON, YAML 등) 간 변환을 도와주는 웹 도구. 다양한 환경 설정 포맷을 손쉽게 상호 변환할 수 있음.
  - **분류**: 환경 설정 / 포맷 변환 / 유틸리티
  - **레벨**: 전 레벨 공통
  - **비고**: 설치 불필요, 브라우저에서 바로 사용 가능

- [TableConvert – Excel to JSON](https://tableconvert.com/ko/excel-to-json)
  - **설명**: 엑셀/테이블 데이터를 JSON 포맷으로 손쉽게 변환해 주는 웹 도구. 컬럼 헤더와 행 데이터를 인식해 구조화된 JSON으로 만들어 줌.
  - **분류**: 포맷 변환 / 데이터 가공 / 유틸리티
  - **레벨**: 전 레벨 공통
  - **비고**: 다양한 포맷 간 상호 변환 지원, 설치 불필요, 브라우저에서 사용 가능

---

## 📝 메모 (선택)

- 기타 포맷 변환기, 인코딩/디코딩 도구, JSON/UUID/암호화 관련 유틸 등을 여기에 함께 정리할 수 있습니다.
